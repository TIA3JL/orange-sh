Get the script:
```
wget https://raw.githubusercontent.com/everydayikillmylinux/orange-sh/main/orange.sh
```

```bash
git clone https://github.com/everydayikillmylinux/orange-sh.git && cd orange-sh
```

```
chmod +x orange.sh
./orange.sh
```
